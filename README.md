To define loader set appsetting.json VendorSettings.LoaderType to  FileLoader or SqlServerLoader.
Start Vendor.API first (at `http://localhost:5138`)
swagger UI will open at http://localhost:5138/swagger/index.html

run UI:
npm start
app opens at `http://localhost:4200`.

Backend code:
Vendor.API - web api application
Vendor.Logic - application logic

UI:
Vendor.UI - generated.
